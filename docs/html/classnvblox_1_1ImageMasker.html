◆ occlusion_threshold_m() [1/2]

float nvblox::ImageMasker::occlusion_threshold_m ( ) const

A parameter getter The occlusion threshold parameter associated with the image splitter.

A point is considered to be occluded on the mask image only if it lies more than the occlusion threshold behind the point occluding it. Occluded points are always assumed to be unmasked.

Returns
the occlusion threshold in meters
