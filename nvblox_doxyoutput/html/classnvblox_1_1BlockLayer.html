template<typename _BlockType >
bool nvblox::BlockLayer< _BlockType >::gpu_layer_view_up_to_date_
mutableprotected

GPU Hash NOTE(alexmillane):

  • This has is subservient to the CPU version. The layer has to copy the hash to GPU when it is requested.
  • Cached such that if no blocks are allocated between requests, the GPULayerView is not recopied.
  • Lazily allocated (space allocated on the GPU first request)
  • The "mutable" here is to enable caching in const member functions.
